# Système Général - Des Récits Discordants

## Concepts Fondamentaux

### Les Dés (dD)
- **Dés Discordants (dD)** : Dés à 3 faces : `+`, `-`, `0`
- **Résultat d'un jet** : Nombre de `+` moins le nombre de `-`
- **Égalité** : Même nombre de `+` et `-`

### Les Épreuves
- **Épreuve Possible** : Action avec chance d'échec
- **Épreuve Impossible** : Action sans chance de succès (sauf critique)
- **Niv d'Épreuve** : Difficulté de l'action (-5 à +10+)

### Jet de Compétence
- **Base** : Niv d'Aptitude + Niv de Compétence + Dés de Compétence
- **Modificateurs** : Situation, équipement, Souffrances, etc.
- **Succès** : Résultat ≥ Niv d'Épreuve
- **Échec** : Résultat < Niv d'Épreuve

### Échecs Critiques et Succès Critiques
- **Échec Critique** : Tous les dés sont `-` → 5 Marques d'expérience
- **Succès Critique** : Tous les dés sont `+` → Effet spécial

### Personnages
- **PI (Personnage Inspiré)** : Personnage joueur
- **PNI (Personnage Non-Inspiré)** : Personnage non-joueur
- **Éveilleur** : Maître de jeu
- **Révélateur** : Concept métaphysique (perspective objective)

## Niveaux et Échelles

### Distinction
- **Distinction** : Capacité à percevoir et exploiter la matrice narrative
- Calculée comme la somme de tous les Niv d'Attributs
- Détermine le Niv de Gouvernance des Titres

### Niveaux de Temps (NdT)
- **Souffle** : 1 seconde
- **Clins** : 1/3 seconde
- **Respiration** : 5 secondes
- **Minute** : 60 secondes
- **Heure** : 60 minutes
- **Jour** : 8 heures de Labeur
- **Pentaine** : 5 jours
- **Saison** : 30 jours de Labeur
- **Année** : 8 Saisons
- **Cycle** : Variable selon les Peuples

## Types de Jets

### Jet Précisé
- Prendre du temps ("Faire 0") = pas de jet, résultat basé sur les niveaux
- Plus de temps = difficulté réduite
- Moins de temps = difficulté augmentée

### Jet Risqué
- Déclarer un seuil d'échec critique pour bonus
- Plus le seuil est bas, plus le bonus est élevé

### Jet Sacrifié
- Réussir sur égalité ou échec critique en acceptant une pénalité
- "Sacrifice Aveugle" : pénalité déterminée par l'Éveilleur

### Jet de Groupe
- Jet individuel modifié par l'Ambiance du groupe
- Résultats moyennés

### Jet d'Audace
- Un individu tente une action de groupe avec pénalités
- Récompenses potentielles plus élevées

## Perspectives et Philosophie

### Perspectivisme
- Chaque perspective a une validité objective dans son contexte
- La multiplicité des perspectives enrichit la compréhension
- Pas de relativisme absolu, mais une objectivité contextuelle

### Les Récits
- Chaque personnage a un **Récit** unique
- Le Récit détermine la Caste et les avantages narratifs
- Double les dés CT dans les situations pertinentes
